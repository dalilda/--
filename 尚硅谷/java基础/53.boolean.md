# boolean只能取true false
```java
class Test{
    public static void main(String args[]){
        boolean BB = true;
        boolean BBB = false;
    }
}
```

# 什么时候用boolean？
条件判断、循环结构
```java
class Test{
    public static void main(String args[]){
        boolean isMarried = true;
        if (isMarried){
            System.out.println("你就不能参加\"单身\"party了\n很遗憾");
        }else{
            System.out.println("你可以找个女朋友");
        }
    }
}
```
