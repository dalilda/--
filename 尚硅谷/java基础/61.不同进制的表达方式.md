# 进制分为二进制 八进制 十进制 十六进制
> 二进制 0b或0B开头
> 八进制 0开头
> 十六进制 0x或0X开头

```java
class Test{
    public static void main(String args[]){
        int num1 = 0b110;
        int num2 = 110;
        int num3 = 0123;
        int num4 = 0x110;
        System.out.println("num1:"+num1);
        System.out.println("num2:"+num2);
        System.out.println("num3:"+num3);
        System.out.println("num4:"+num4);
    }
}
```
