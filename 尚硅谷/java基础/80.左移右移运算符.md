# 位运算符
<< >> >>> & | ^ ~ 

## << 左移
1101=(2^3+2^2+2^0) 左移两位后
110100 =2*2*(2^3+2^2+2^0)
所以每左移一位相当于乘2

# 说明
1. 位运算符操作的都是整型
2. << :在一定范围内,每向左移一位,相当于乘2
3. >> :在一定范围内,每向右移一位,相当于除2
