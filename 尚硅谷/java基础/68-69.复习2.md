1. 变量按数据类型分,注意事项
long类型结尾要加L
float类型结尾要加F
整型默认是int类型
浮点型默认是double类型
char通常使用一对'',里面可以只有一个字母;可以是转义字符;可以是unicode值。
char c = 97;      //里面存的是'a'
char c = 48;      //里面存的是'0'
char c = '0';
int i1 = (int)c;  //i1等于48。其实就是ascii码中'0'的序号
